# 🎬 **Nervflix**

Una plataforma de streaming inspirada en Netflix, dedicada exclusivamente al universo de **Neon Genesis Evangelion**.
Explora la serie original y las películas *Rebuild* en una experiencia cinematográfica inmersiva.

[![Nervflix Live Demo](https://img.shields.io/badge/Nervflix-Live%20Demo-red?style=for-the-badge\&logo=react)](https://nervflix-demo.example.com)

---

## ✨ Características

* 🎥 **Catálogo completo** – Serie original y películas *Rebuild*
* 📱 **Diseño responsivo** – Optimizado para móvil, tablet y desktop
* 🎭 **Reproductor integrado** – Iframes embebidos con navegación fluida
* 📚 **Guía del universo** – Línea de tiempo, personajes, ángeles y más
* 🎨 **Estilo Netflix** – UI oscura y elegante con UX intuitiva
* ⚡ **Transiciones suaves** – Navegación SPA sin recargas

---

## 🛠️ Stack Tecnológico

* **Frontend**: React 18 + TypeScript
* **Build Tool**: Vite
* **Estilos**: Tailwind CSS + `shadcn/ui`
* **Ruteo**: React Router DOM
* **Estado**: TanStack Query
* **Componentes UI**: Radix UI + Lucide React
* **Animaciones**: Tailwind CSS Animate

---

## 📁 Estructura del Proyecto

```
src/
├── components/     # Componentes reutilizables
│   ├── ui/         # Componentes base de shadcn/ui
│   ├── timeline/   # Guía del universo
│   └── *.tsx       # Header, Hero, etc.
├── data/           # Datos estáticos
│   ├── movies/     # Episodios y películas
│   └── timeline/   # Ángeles, personajes, etc.
├── pages/          # Páginas principales
├── types/          # Tipos de TypeScript
├── hooks/          # Custom Hooks
└── lib/            # Utilidades/config
```

# 🎬 **Nervflix**

A Netflix-inspired streaming platform dedicated exclusively to the world of **Neon Genesis Evangelion**.
Experience the original series and the *Rebuild* films in an immersive, cinematic interface.

[![Nervflix Live Demo](https://img.shields.io/badge/Nervflix-Live%20Demo-red?style=for-the-badge\&logo=react)](https://nervflix-demo.example.com)

---

## ✨ Features

* 🎥 **Complete Catalog** – Original TV series and *Rebuild* films
* 📱 **Responsive Design** – Optimized for mobile, tablet, and desktop
* 🎭 **Built-in Player** – Embedded iframes for smooth playback
* 📚 **Universe Guide** – Timeline with characters, angels, organizations
* 🎨 **Netflix-style UI** – Dark, elegant interface with a familiar UX
* ⚡ **Fluid Navigation** – Seamless transitions between sections

---

## 🛠️ Tech Stack

* **Frontend**: React 18 + TypeScript
* **Build Tool**: Vite
* **Styling**: Tailwind CSS + `shadcn/ui`
* **Routing**: React Router DOM
* **State Management**: TanStack Query
* **UI Components**: Radix UI + Lucide React
* **Animations**: Tailwind CSS Animate
